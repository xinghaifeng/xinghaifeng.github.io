  • AWS EC2虚机介绍

AWS EC2是亚马逊云主打产品，也是用户使用最广泛的产品，相当于传统机房的一台物理机器，可以配置多硬盘、多网卡等等

闲言碎语不多讲，直接上图创建一台EC2：
1、登陆到控制台，选择EC2服务

2、启动EC2实例，开始创建EC2

3、选择EC2实例操作系统，这里我们默认AWS提供的Linux

4、选择EC2机型配置，可以根据需求、价格等因素选择

5、配置详细信息，启动机器数量、网络、IAM、监控

6、配置存储，往EC2上挂硬盘

7、配置安全组策略，开放哪些端口，设置白名单

8、审查启动实例 ，仔细检查一遍配置

9、创建PEM密钥对 ，一定要保存好，以后登录全靠它，没有就煞笔了

10、分配静态IP地址，如果创建EC2时候没有分配，那么实例创建后最好分配一下 ，否则重启实例IP会漂移的！

11、EC2创建完成后仍然可以管理安全组，设置白名单

12、使用PUTTY登录配置，下载puttygen将pem文件生成ppk文件（1）

13、使用PUTTY登录配置（2）

14、连接后，输入ec2-user敲回车，无需密码，登陆啦！

15、更多机型供选择:https://aws.amazon.com/cn/ec2/instance-types/
