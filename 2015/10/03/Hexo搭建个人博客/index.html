Hexo搭建个人博客

By 邢海峰

发表于 2015-10-03

文章目录

2010年以前一直在JavaEye写博客，然后比较懒 ，不怎么写博客了。WordPress还有维护成本，hexo是一个基于Node的博客框架，支持Markdown语法简洁太有牌面了，使用简单而且可以部署在GitHub上 ，没有运维成本 ，唯一不足是GitHub屏蔽百度爬虫，无法被百度爬虫收录。

步骤

1、安装Node客户端
https://nodejs.org/en/download/

2、安装Git

http://code.google.com/p/msysgit/

3、安装Hexo

第一步：npm install hexo-cli -g

第二步：新建blog目录，然后在Git Bash执行hexo init blog 生成网站所需文件

第三步：执行hexo g 再执行hexo s、浏览器输入localhost:4000可以看到博客主页搭建完成

第四步：博客主题下载 git clone https://github.com/wuchong/jacman.git themes/jacman

第五步：修改Hexo目录下的config.yml配置文件中的theme属性，将其设置为jacman。同时请设置stylus属性中的compress值为true

注意：Hexo有两个config.yml文件，一个在根目录，一个在theme下，此时修改的是在根目录下的

本地查看调试

$ hexo g #生成

$ hexo s #启动本地服务，进行文章预览调试

4、将独立域名与GitHub Pages的空间绑定

GitHub Pages的设置

方法一：在Repository的根目录下面，新建一个名为CNAME的文本文件，里面写入你要绑定的域名，比如xinghf.com。

方法二：到我的github仓库，点击右下角的「Download ZIP」，下载源文件，解压，找到CNAME文件，用记事本打开，将cnfeat.com修改成你的域名，放进Hexo\source目录下，用hexo命令提交上去。

在新网注册域名，域名A记录指向GitHub ,进行域名绑定

5、集成评论功能+CNZZ

第一步：静态博客要使用第三方评论系统，hexo默认集成的是Disqus，建议用多说评论
第二步：注册CNZZ ，修改模板配置文件，监控网站流量

6、Hexo命令

常用命令：

hexo new “postName” #新建文章
hexo new page “pageName” #新建页面
hexo generate #生成静态页面至public目录
hexo server #开启预览访问端口 默认端口4000
hexo deploy #将.deploy目录部署到GitHub

常用复合命令：

hexo d -g #生成加部署
hexo s -g #预览加部署
