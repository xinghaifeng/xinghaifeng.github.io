《一个人的电商：树莓派+NodeJS+MongoDB搭建微信公众平台》

By 邢海峰

发表于 2015-10-08

文章目录
  1. 1. 前言
    1. 1.1. 步骤：

最近读了许晓辉的书 《一个人的电商：运营策略与实操手记》深受启发，技术、运营、市场、营销、运维缺一不可，我崇拜个人英雄主义、单打独斗。因此再扩展思维写一篇《一个人的电商：树莓派+NodeJS+MongoDB搭建微信公众平台》

前言

用树莓派搭建Linux服务器，通过ngrok将内网映射到公网上面，这样就可以在公网访问我们用树莓派搭建的网络服务
NodeJS+MongoDB搭建微信公众平台,从产品设计、需求分析、开发、测试一个人全部搞定，省去沟通成本

步骤：

1、注册微信开放平台
mp.weixin.qq.com

2、获取TOKEN
在公众平台个人首页的右下角点击“开发者中心” 如图：

看一下技术文档，如何获取TOKEN一会NodeJS搭建完成后即可实际操作

3、树莓派安装系统镜像

安装文档及镜像，下载链接：http://pan.baidu.com/s/17k6a 密码：gvox

4、树莓派安装MongoDB
下载地址：https://www.mongodb.org/downloads

5、树莓派安装NodeJS、NPM、GIT、截屏方法
NodeJS下载地址： https://nodejs.org/en/download/ 注意一定要下载ARMv7版本，如图：




npm安装参考：http://npm.taobao.org/

GIT安装:sudo apt-get install git

树莓派截屏：
 sudo apt-get install scrot      
 sudo scrot -s

6、ngrok安装

7、用NodeJS搭建微信菜单栏

8、用NodeJS开发业务功能
获取TOKEN、自定义菜单栏、新闻发布、产品发布、产品追踪

9、微信公众平台功能
优粮科技为佳沃开发公众号 ，产品生产全流程跟踪、产品展示、营销活动等。
