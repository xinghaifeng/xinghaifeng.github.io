GitBook 是一个基于 Node.js 的命令行工具，可使用 Github/Git 和 Markdown 来制作精美的电子书。
本文将简单介绍如何安装、编写、生成、发布一本在线图书。

GitBook支持输出多种文档格式，如：

静态站点：GitBook默认输出该种格式，生成的静态站点可直接托管搭载Github Pages服务上；PDF：需要安装gitbook-pdf依赖；eBook：需要安装ebook-convert；
单HTML网页：支持将内容输出为单页的HTML，不过一般用在将电子书格式转换为PDF或eBook的中间过程；JSON：一般用于电子书的调试或元数据提取。

GitBook项目地址

GitBook项目官网：http://www.gitbook.io

GitBook Github地址：https://github.com/GitbookIO/gitbook

Gitbook是从NMP安装的，命令行：

$ npm install gitbook -g

安装完之后，你可以检验下是否安装成功：

$ gitbook -V
0.4.2

如果你看到了与上面类似的版本信息，则表示你已成功完装上了Gitbook。

检出gitbook模板项目

$ git clone https://github.com/waylau/gitbook.git

初始化目录结构

$ gitbook init

Build 编译

$ gitbook build ./

查看默认生成的文档

打开该目录，右键运行 index.html 即可看到 gitbook 生成的html文档
